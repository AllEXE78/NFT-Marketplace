# NFT Marketplace Landing Page

Адаптивный лендинг для NFT-маркетплейса, сверстанный с нуля по макету из Figma. Проект создан в учебных целях для отработки навыков современной веб-разработки, семантической верстки и доступности.

Проект основан на макете NFTMarketplace Community из Figma.
