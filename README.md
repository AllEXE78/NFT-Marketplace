# NFT Marketplace

Одностраничный лендинг, свёрстанный по макету Figma.

## 🛠 Стек
- HTML5 (БЭМ)
- CSS3 (Flexbox, переменные, анимации)
- Git и GitHub Pages

## 🎯 Что сделано
- Семантичная вёрстка по БЭМ
- Адаптивная hero-секция с позиционированием элементов
- Кастомные световые эффекты на CSS
- Интерактивные элементы (hover-состояния)

## 📂 Запуск
Откройте `index.html` в браузере или перейдите по ссылке:
[https://allexe78.github.io/NFT-Marketplace/](https://allexe78.github.io/NFT-Marketplace/)
