# NFT Marketplace Landing Page

Адаптивный лендинг для NFT-маркетплейса, сверстанный с нуля по макету из Figma. Проект создан в учебных целях для отработки навыков современной веб-разработки, семантической верстки и доступности.
**Живое демо:** https://allexe78.github.io/NFT-Marketplace/

## Технологии

- **HTML5** (Семантическая разметка, доступность a11y)
- **CSS3** (Flexbox, Grid, CSS Variables, адаптивность)
- **JavaScript** (Базовая интерактивность)
- **GitHub Pages** (Деплой и хостинг)

## Особенности

- ✅ Полностью адаптивный дизайн (Mobile First / Desktop First)
- ✅ Оптимизированная загрузка ресурсов (WebP/AVIF, ленивая загрузка)
- ✅ Внимание к доступности (ARIA-атрибуты, фокус-состояния, контраст)
- ✅ Чистая структура проекта и BEM-подобный нейминг классов
- ✅ Плавные анимации и hover-эффекты
